## API
Specs:  
- Framework: ASP.NET Core 5.0 Web API Framework
- ORM: EF Core 5.0
- DB: PostgreSQL
- Auth: Lightweight JWT Authentication & Endpoint authorization

## Client
Specs:  
- Framework: React
- UI Library: Material UI
- Type checking: Typescript
- State mgmt: Redux
